IMPORTANT!
Garry's Mod 9 requires that Source SDK Base 2006 is installed.

If not, Garry's Mod will not run and will throw an error.

LOAD TIMES!

The first time you run a map it will take a long time to load. This is because it creates a cache file which speeds up loading later on. Sorry, load times aren't really my fault!

Getting help..
If you're stuck you should look for help in these places:
  • GMod Website - Make sure you're using the most recent version.
  • GMod Wiki - The GMod user's bible.
  • GMod Forums - Get help, download new maps, models and mods.
